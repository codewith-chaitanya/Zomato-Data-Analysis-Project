# Zomato Restaurant Data Analysis Project

This repository contains an end-to-end data analysis project using Python, Pandas, Matplotlib, and Seaborn to uncover patterns within restaurant data.

## Project Contents
- `Zomato_Data_Analysis_Project.ipynb`: Complete Jupyter Notebook containing Data Loading, Cleaning, EDA, 6 Visualizations, and an Insights Report.

## Key Insights Discovered
1. Premium priced restaurants hold consistently higher ratings.
2. Over 74% of listed restaurants have yet to adopt online delivery configurations.
3. High customer engagement strongly correlates with an aggregate rating score between 3.5 and 4.5.
